Metrics Explained

  • FCP: First Contentful Paint — time until the first text/image appears.
  • LCP: Largest Contentful Paint — time until the main content is visible.
  • CLS: Cumulative Layout Shift — visual stability score (lower is better).
  • TBT: Total Blocking Time — how long scripts block interaction.
  • Speed Index: how quickly content is visually displayed.
